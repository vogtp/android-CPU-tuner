Configuration

Configurations are backups of the profiles, triggers and virtual governors.
  • Save configurations
    When checked cpu tuner will automatically save the current configuration.
There are two types of configurations:
  • Youruserconfigurations
    These are the configurations you use(d). They can be saved automatically on each change or manually.
  • System configurations
    These are the configurations provided by the system.
Configurations can be used to:
  • Apply different settings depending on time
    Configurations can be automatically loaded at a given time, in order to implement different settings for (e.g.) day and night.
  • Backup and restore the current settings
    After a reinstall or to test the behavior of different settings.
